# 雨窗阁 | Rain Window Study

一个为文字创作者设计的沉浸式写作环境。

## 功能
- 🌧️ 动态雨景背景 + 氛围音效
- ✍️ 专注写作模式 + 实时字数统计
- 🍅 番茄钟时间管理
- 📓 手账本（支持手绘 + 贴纸）
- 📊 小说市场数据分析
- 💾 本地数据存储 + 备份导出

## 技术
- 纯前端实现（HTML/CSS/JS）
- Canvas 粒子动画（光标漩涡、打字特效）
- Web Audio API 生成氛围音乐
- LocalStorage 持久化用户数据
- 响应式布局（手机/平板/电脑）
